#### 1. 介绍
- 为子系统的一组接口提供了一个高层接口
- 使用者使用这个高层接口

#### 2. 场景
```
function bingEvent(ele, type, selector, fn) {
    if (fn == null) {
        fn = selector;
        selector = null;
    }
}

//调用
bingEvent(elem, "click", "#div", fn);
bingEvent(elem, "click", fn);
```

#### 3. 设计原则校验
- 不符合单一职责原则和开放封闭原则，因此谨慎使用，不可滥用
